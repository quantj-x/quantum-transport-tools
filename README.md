# Cryogenic Quantum Transport Measurement GUI

A modular Python GUI framework for low-temperature quantum transport measurements.

This project is developed for cryogenic electrical transport experiments on mesoscopic devices and two-dimensional quantum materials. It provides an integrated graphical interface for instrument control, parameter sweeping, real-time data visualization, and structured data logging. The current implementation is particularly suited for dual-gate mapping measurements using lock-in amplifier readout and DAQ-based voltage acquisition.

The framework emphasizes experimental reliability, readability, and reproducibility. Measurement parameters, raw data, converted resistance values, and metadata are saved in a structured format to facilitate later analysis, figure preparation, and long-term data management.

Although the code was originally designed for a specific laboratory setup, the architecture is intended to be extensible to other instruments and measurement configurations.

## Features

- Graphical user interface for cryogenic quantum transport measurements
- Dual-gate voltage sweep and two-dimensional resistance mapping
- Lock-in amplifier readout for low-frequency AC transport measurements
- DAQ-based voltage acquisition and real-time signal monitoring
- Real-time line plots and 2D resistance color maps
- Automatic conversion from measured voltage signals to resistance values
- Structured data saving with experimental parameters and metadata
- Configurable sweep range, step size, delay time, excitation current, and channel assignment
- Modular code structure for instrument control, measurement logic, plotting, and data handling
- Designed for extensibility to different laboratory setups and instruments

## Typical Measurement Configuration

The program is designed for a typical low-temperature four-terminal transport configuration. A low-frequency AC excitation is generated by a lock-in amplifier and converted into an approximately constant current through a large series resistor. The voltage responses, such as longitudinal and Hall voltage signals, are acquired through DAQ channels or lock-in outputs. The software then converts the measured voltages into resistance values using user-defined calibration factors.

Typical applications include gate-dependent longitudinal and Hall resistance mapping, dual-gate transport measurements, low-frequency lock-in measurements, and parameter sweeps under cryogenic conditions.

## Safety Notice

This software may directly control laboratory instruments and apply voltages or currents to sensitive cryogenic devices. Users should carefully verify all instrument addresses, wiring configurations, voltage limits, current limits, sweep ranges, delay times, and calibration factors before running any measurement.
